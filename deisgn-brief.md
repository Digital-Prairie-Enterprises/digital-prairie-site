# Digital Prairie Enterprise - Website Design Brief

## Brand Style
Cyber-cowboy asthetic: earth tones plus neon accents.
Rugged meets futuristic. Lonesome Dove meets Neuromancer.
Clean, readable typography - prefer serif fonts.
Logo colors drive palette.

## Pages
- Home
- About
- Products
- Contact
- Blog

## Requirements
- Responsive layout
- Shared header & footer
- CSS variables for color palette extracted from logo
- use assets/logo.jpeg for palette inspiration
- use assets/headshot.jpg for "Our People" section
- Keep HTML clean and semantic
- Keep CSS in styles.css
- Keep JS minimal

## Blog
Static HTML pages are fine.
Each post can be a new HTML file in /blog
